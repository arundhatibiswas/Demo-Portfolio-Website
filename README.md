# Demo Portfolio Website

> A clean, responsive **portfolio website** template to showcase projects, skills, and contact information.

## 🎯 Overview

A modern portfolio site built with TypeScript, designed to present work and personal branding in a fast, mobile-friendly layout. Easy to customize for any developer or creative.

## ✨ Features

- Fully responsive, mobile-first design
- Sections for About, Projects, Skills, and Contact
- Smooth, modern UI with reusable components
- Optimized for performance and SEO

## 🛠️ Tech Stack

- **TypeScript**
- **HTML5** & **CSS3**
- Component-based front-end architecture

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/arundhatibiswas/Demo-Portfolio-Website.git
cd Demo-Portfolio-Website

# Install dependencies
npm install

# Run the development server
npm run dev
```

> Adjust commands to match the project's actual package scripts.

## 🎨 Customization

Update the content (name, bio, projects, links) and theme colors in the relevant config/component files to make it your own.
